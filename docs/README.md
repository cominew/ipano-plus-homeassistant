# iPano Plus — Documentation

This documentation covers installation, configuration, services, events, example automations, developer notes and troubleshooting for the iPano Plus Home Assistant integration.

## Installation

Via HACS (recommended)
1. In HACS → Integrations → ⋯ (top right) → Custom repositories
2. Add repository URL: `https://github.com/Cominew/ipano-plus-homeassistant`
3. Category: Integration
4. Install the integration from HACS
5. Restart Home Assistant

Manual installation
1. Download the latest release ZIP or clone the repository.
2. Copy the folder `custom_components/ipano_plus/` into your HA configuration directory: `config/custom_components/ipano_plus/`
3. Restart Home Assistant
4. Add the integration via **Settings → Devices & Services → Add Integration → iPano Plus**
