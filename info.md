# iPano Plus Home Assistant Integration

This integration adds support for 6‑inch iPano Plus in‑wall touchscreen panels (buttons, backlights, relays, proximity sensor and screen control).

![HACS](https://my.home-assistant.io/badges/hacs_repository.svg)

Install via HACS (Integration category) or manually copy `custom_components/ipano_plus/` into your Home Assistant config.
